part-A

1. Directly run MyTopo.py file in part-A folder, also contains the PDF explanation. It will generate the static routing tables.

part-B

1. Directly run myRIP.py in part-B folder, also contain the PDF explanation,the folder should contain all the r1, r2, r3, r4, h1 and h2 folders, which contains the necessary conf files for bird execution.

Part-C

1. Directly run MyIperf.py in part-C folder, also contain the PDF explanation,the folder should contain all the r1, r2, r3, r4, h1 and h2 folders, which contains the necessary conf files for bird execution. Currently the .py file is set for buffer size of 25 mbits, you can change that to the other cases, if you want to test it.
